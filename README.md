GSImage - Image handling for PHP
================================

GSImage provides generell image functions via various drivers for php.

It supports the following drivers:
* GD
<!--
* Imagick
* convert (via exec)
* PHP native
-->

Requirements
------------

PHP 5.3.3 or above
